# Proyek Analisis Data: Beijing Air Quality Analysis By Rafi Geovazi 🍃

## Setup Environment
```bash
pip install -r requirements.txt
```

## Run Notebook
1. Buka `notebook.ipynb` di Jupyter Notebook / Google Colab
2. Jalankan semua cell secara berurutan
3. File `dashboard/main_data.csv` akan otomatis ter-generate

## Run streamlit app
```bash
cd dashboard
streamlit run dashboard.py
```
